# Парсер вакансий

Скрипт предназначен для сбора статистики вакансий по языкам программирования.
Собирает статистику с сайтов [hh.ru](https://hh.ru/) и [superjob.ru](https://www.superjob.ru/)

## Что нужно сделать перед запуском
Проверьте версию Python:
```bash
python --version
```
- Должна быть версия 3.8 или выше

Установите необходимые библиотеки:
```bash
pip install -r requirements.txt
```
**Часть настроек проекта берётся из переменных окружения.**   
**Чтобы их определить, создайте файл .env (точка в начале обязательна!) рядом с main.py**   
**и запишите туда данные в таком формате: ПЕРЕМЕННАЯ=значение.**
```bash
AREA_HH=1   # ID города для hh.ru  Пример 1 (Москва)
TOWN_SJ=4   # ID города для superjob.ru  Пример 4 (Москва)
```
**Важно:** ID города для каждого сайта свой не перепутайте

```bash
PERIOD=0  # можно не указывать
Период за которы нужно собрать статистику 
Возможные значения:
        0 — за всё время.
        1 — за последние 24 часа.
        3 — за последние 3 дня.
        7 — неделя (Для SuperJob это максимум)
        30 — за последний месяц (Для hh.ru это максимум).
```

```
PROGRAMMING_LANGUAGES=Python,Java,Javascript,C#,Ruby,C++,GO,TypeScript
```
**Ключевые слова для поиска в каждом сайте.**   
**В примере используется список языков программирования.**

```bash
SJ_KEY=ваш_секретный_API_ключ 
```
API-ключ необходим для работы с API SuperJob.

### Получите токен `SJ_KEY`
- Зарегистрируйтесь на сайте [SuperJob](https://api.superjob.ru/)   
- Найдите там свой **API ключ** (обычно в личном кабинете или настройках профиля)   
- Скопируйте его

## Как запустить?
**Скачиваем код**
**Открываем CLI** - Нажимаем Win + R, вводим: cmd
```bash
   cd путь/к/папке/с/кодом
```
Запускаем:
```bash
python main.py
```
